# Maid Nana 用户手册
## 命令列表
> [] 代表可选参数, <> 代表必选参数, ...代表多个参数

> 所有命令需要私聊机器人发送
### 帮助
#### 格式
```
帮助
```
#### 描述
查看帮助列表

### 新建公告
#### 格式
```
新建公告 [公告别名]
```
#### 描述
新建一个公告, 创建后会自动选择

可指定 [别名](#别名) 便于记忆

### 选择公告
#### 格式
```
选择公告 <公告 UUID | 公告别名>
```
#### 描述
为后续命令指定公告

### 删除公告
#### 格式
```
删除公告
```
#### 描述
删除 [选定](#选择公告) 的公告

### 公告列表
#### 格式
```
公告列表
```
#### 描述
查看公告列表

## 别名
别名是一个不可重复的标识符, 用于替代 UUID 帮助记忆.
