# Maid Nana 用户手册
## 命令列表
> [] 代表可选参数, <> 代表必选参数, ...代表多个参数

> 所有命令需要私聊机器人发送
### 帮助
#### 格式
```
帮助
```
#### 描述
查看帮助列表

### 新建公告
#### 格式
```
新建公告 [公告别名]
```
#### 描述
新建一个公告, 创建后会自动选择

可指定 [别名](#别名) 便于记忆

### 选择公告
#### 格式
```
选择公告 <公告 UUID | 公告别名>
```
#### 描述
为后续命令指定公告

### 删除公告
#### 格式
```
删除公告
```
#### 描述
删除 [选定](#选择公告) 的公告

### 公告列表
#### 格式
```
公告列表
```
#### 描述
查看公告列表

### 设置群
#### 格式
```
设置群 <群号1> [群号2]...
```
#### 描述
设置[选定](#选择公告)公告发送的群, 可设置多个

### 取消群
#### 格式
```
取消群 <群号1> [群号2]...
```
#### 描述
取消设置[选定](#选择公告)公告发送的群, 可取消多个

### 设置纯文本公告
#### 格式
```
纯文本公告
<公告体(多行)>
```
#### 描述
为[选定](#选择公告)公告设置纯文本公告体

### 设置模板公告
#### 格式
```
模板公告 <模板UUID | 模板别名>
<变量1> = <值1>
[变量2] = [值2]
...
```
#### 描述
为[选定](#选择公告)公告设置模板公告体
> 创建模板使用[创建模板命令](#新建模板)

#### 样例
```
模板公告 上课提醒
name = 离谱数学
meeting_number = 114514
```

### 开启公告
#### 格式
```
开启公告
```
#### 描述
开启[选定](#选择公告)公告的定时发布

### 禁用公告
#### 格式
```
禁用公告
```
#### 描述
禁用[选定](#选择公告)公告的定时发布

禁用后只能通过[发送公告](#发送公告)命令手动发送


### 预览
#### 格式
```
预览
```
#### 描述
预览[选定](#选择公告)公告发送后结果 (会展示模板变量替换后的结果)

### 设置公告变量
#### 格式
```
设置变量
<变量1> = <值1>
[变量2] = [值2]
...

```
#### 描述
为[选定](#选择公告)的**模板**公告设置变量

#### 样例
```
设置变量
name = 离谱数学
meeting_number = 114514
```

### 取消设置公告变量
#### 格式
```
取消变量 <变量1> [变量2]...
```
#### 描述
为[选定](#选择公告)的**模板**公告取消设置变量

### 查看公告
#### 格式
```
查看公告
```
#### 描述
查看[选定](#选择公告)公告的信息

### 发送公告
#### 格式
```
发送公告
```
#### 描述
手动发送[选定](#选择公告)公告

### 新建触发器
#### 格式
```
新建触发器 <cron表达式>
```
#### 描述
为[选定](#选择公告)的**模板**公告添加[cron表达式](#Cron表达式)的触发器

### 删除触发器
#### 格式
```
删除触发器 [触发器 UUID]...
```
#### 描述
为[选定](#选择公告)的**模板**公告删除触发器, 若不指定触发器则删除全部

### 新建模板
#### 格式
```
新建模板 [模板别名]
<模板体(多行)>
```
#### 描述
新建模板, 使用 `$` 定义变量, 如 `$name$`

### 删除模板
#### 格式
```
删除模板 <模板 UUID | 模板别名>
```
#### 描述
删除模板

### 修改模板
#### 格式
```
修改模板 <模板 UUID | 模板别名>
<模板体(多行)>
```
#### 描述
修改指定的模板内容

### 查看模板
#### 格式
```
查看模板 <模板 UUID | 模板别名>
```
#### 描述
查看模板信息

### 模板列表
#### 格式
```
模板列表
```
#### 描述
查看模板列表


# 别名
别名是一个不可重复的标识符, 用于替代 UUID 帮助记忆.
